The first step to creating an image is to create a container and fill it with some data.

First we'll create a container with a single new file based off of the busybox base image:

sudo docker run busybox sh -c "echo a > newfile"

The container will immediately terminate (because its one command is sh -c "..."), so we'll use docker ps -n 1 to list it: 

sudo docker ps -n 1
CONTAINER ID  IMAGE                             COMMAND               CREATED         STATUS                     PORTS   NAMES
63f8105cf522  docker.io/library/busybox:latest  sh -c echo a > ne...  10 seconds ago  Exited (0) 10 seconds ago          cranky_bardeen
